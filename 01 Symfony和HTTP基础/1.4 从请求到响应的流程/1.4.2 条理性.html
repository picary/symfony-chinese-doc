1.4.2 条理性

在前端控制器里面，你必须知道哪一段程序需要执行，什么样的内容应该返回。需要明白的是，你需要检查进来的URI，并根据不同的值执行你代码中不同的部分。如果这样代码可能很快就会变的非常丑陋。

// index.php
use Symfony\Component\HttpFoundation\Request;
use Symfony\Component\HttpFoundation\Response;

$request = Reqeust::createFromGlobals();
$path = $request->getPathInfo(); // 被请求的URI

if(in_array($path, array('', '/'))){
$response = new Response('Welcome to the homepage.'); } elseif ('/contact' === $path) {
$response = new Response('Contact us.'); } else {
$response = new Response('Page not found.', Response::HTTP_NOT_FOUND); }
$response->send();

解决这个问题将会是困难的。幸运的是，设计Symfony的目的就在与此。
